## 一、核心视觉语言

### 1.1 风格定义：赛博几何风
- **一切元素**由基础或变形的几何体构成（立方体、球体、棱锥等）
- 融合**赛博朋克的霓虹光效**与**数据流视觉语言**
- **纯俯视2D正交视角**，无透视变形

### 1.2 体验核心
- 服务于"割草"爽快感
- 夸张视觉反馈、清晰敌我辨识
- 绚烂但有序的屏幕特效
- 流畅飘逸的动作节奏

### 1.3 技术栈
- **平台**：微信小游戏（移动端）
- **技术**：Spine骨骼动画、Sprite、Tilemap
- **目标帧率**：60fps（可接受30fps）

## 三、属性-形状-色相规范 ⭐重点

**核心原则**：特效必须遵循"属性-形状-色相"严格挂钩

| 属性 | 形状 | 色相 | 示例 |
|-----|------|------|------|
| **🔥 火** | 三角形/锯齿状 | 红/橙/黄 | 火焰飞弹、燃烧特效 |
| **❄️ 冰** | 六边形/雪花状 | 蓝/青/白 | 冰霜箭、冰冻光环 |
| **⚡ 电** | 折线/球形电场 | 蓝紫/亮蓝/白 | 电弧、闪电链 |

### 3.1 粒子形状参考
- **火**：三角形、锯齿状碎片、向上飘升带抖动
- **冰**：六边形雪花、冰晶碎片、缓慢飘落带旋转
- **电**：折线、球形电场粒子、快速闪烁跳跃

### 3.2 材质参考
- **核心**：实心发光色
- **边缘**：明亮光晕
- **拖尾**：半透明同色系渐变

## 四、角色设计规范

### 4.1 玩家角色
- **基础形态**：发光的核心几何体（多面体）
- **成长可视化**：
  - 武器系统（发射物数量、形态、特效）
  - 身体附着物（几何光环、防御护盾）
- **动作设计**：迅捷飘逸，**移动带残影**

### 4.2 敌人设计
- **顶视图轮廓鲜明易辨**
- **分类**：基础型、聚合型、精英型
- **统一死亡反馈**：
  ```
  碎裂 → 粒子化
  ① 碎裂成更小几何体（保持类型对应形状）
  ② 向四周散开带旋转
  ③ 爆散为属性对应颜色粒子
  ④ 渐隐消失
  ```

## 五、特效实现指南

### 5.1 普攻特效分层

```
Layer 1: 发射点
├─ 核心闪烁（Scale 1→1.2→1）
└─ 2-3个属性粒子溢出

Layer 2: 投射物本体
├─ 几何形状（属性对应）
├─ 自发光核心
├─ 边缘光晕
└─ 旋转动画

Layer 3: 拖尾
├─ Trail Renderer
├─ 渐变纹理
└─ Additive混合

Layer 4: 命中反馈
├─ 敌人受击闪白
├─ Camera Shake
├─ 碎裂几何体（3-5个）
└─ 属性粒子爆散
```

### 5.2 Trail Renderer配置
```
Time: 0.1-0.15秒
Width Curve: 前细后粗（刀形状）
Color: 实心色 → 透明渐变
Texture: 简单渐变条
```

### 5.3 粒子性能预算
- **单个技能** < 50个粒子
- **同屏所有技能** < 200个粒子
- **Draw Call** < 20（使用图集合并）

## 六、Shader规范

### 6.1 材质类型
| 元素 | 材质描述 |
|-----|---------|
| **金属地板** | 深蓝金属基底，发光网格线和电路图案 |
| **能量晶体** | 实心核心色，边缘明亮光晕 |
| **几何体核心** | 高光能量体 |
| **飞刃** | 锋利金属质感 |
| **护盾** | 全息能量质感，半透明流动效果 |

### 6.2 发光效果
- **边缘发光**：平台、晶体边缘
- **核心发光**：角色核心、能量单元
- **电弧发光**：电击效果

### 6.3 微信小游戏兼容Shader
| 效果 | 方案 | 性能 |
|-----|------|------|
| 发光 | 自发光 + Additive混合 | ⭐⭐ 低 |
| 溶解 | Alpha Clip + Noise | ⭐⭐ 低 |
| UV动画 | 纹理滚动 | ⭐ 极低 |
| 顶点动画 | Shader修改顶点 | ⭐⭐ 中 |
| 全屏Bloom | 后处理 | ⭐⭐⭐⭐ 高（慎用） |

## 七、光影规范

### 7.1 光照原则
- **统一光源**：上方光源照明
- **平涂着色**：避免复杂阴影
- **环境光**：冷光为主，对比强烈

### 7.2 阴影处理
- **独立物件**：底部微小阴影
- **避免**：侧面阴影、不均匀光照

## 九、AI生成Prompt模板

### 9.1 视角命令（必须）
```
纯粹垂直俯视视角，正交投影，平面2D，90度向下看，无任何透视
```

### 9.2 风格关键词
```
赛博几何风格，低多边形，干净矢量艺术，科幻感
```

### 9.3 技术参数
```
--ar 1:1（独立物件/地砖）
--ar 16:9（场景背景）
--v 6.0
--style raw
```

### 9.4 完整Prompt结构
```
[绝对俯视视角] + [主体描述] + [材质光照] + [色彩方案] + [赛博几何风格] + [技术参数]
```

### 9.5 负面提示
```
避免：透视变形、侧面视角、3D立体效果、写实风格、柔和边缘
```
