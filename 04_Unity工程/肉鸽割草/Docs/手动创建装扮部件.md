# 手动创建装扮部件数据

## 方法：直接在 Unity 中创建

### 步骤 1：创建文件夹
1. 在 Project 窗口中，右键点击 **Assets**
2. 选择 **Create → Folder**
3. 命名为 **Resources**（如果还没有的话）
4. 在 Resources 文件夹内，再创建一个文件夹，命名为 **OutfitParts**

### 步骤 2：创建部件数据文件
1. 右键点击 **Assets/Resources/OutfitParts** 文件夹
2. 选择 **Create → 绒毛几何物语 → 装扮部件数据**
3. 命名为 **bow_red_001**

### 步骤 3：配置数据
选中刚创建的 bow_red_001，在 Inspector 中设置：

| 字段 | 值 |
|------|-----|
| **Part Id** | bow_red_001 |
| **Part Name** | 红色蝴蝶结 |
| **Description** | 热情活泼的红色蝴蝶结 |
| **Category** | Bow |
| **Is Unlocked By Default** | ✓ 勾选 |
| **Offset** | X: 0, Y: 0.3 |
| **Scale** | X: 1, Y: 1 |

### 步骤 4：重复创建
重复步骤 2-3，创建以下部件：

| 文件名 | Part Id | Part Name | Category | Offset Y |
|--------|---------|-----------|----------|----------|
| bow_pink_001 | bow_pink_001 | 粉色蝴蝶结 | Bow | 0.3 |
| hat_beanie_001 | hat_beanie_001 | 毛线帽 | Hat | 0.35 |
| hat_straw_001 | hat_straw_001 | 草帽 | Hat | 0.35 |
| glasses_round_001 | glasses_round_001 | 圆框眼镜 | Glasses | 0.1 |
| glasses_sun_001 | glasses_sun_001 | 墨镜 | Glasses | 0.1 |
| scarf_red_001 | scarf_red_001 | 红色围巾 | Scarf | -0.1 |
| backpack_bear_001 | backpack_bear_001 | 小熊背包 | Backpack | 0 |
| special_wings_001 | special_wings_001 | 天使翅膀 | Special | 0.2 |

### 步骤 5：验证
1. 确保所有文件都在 **Assets/Resources/OutfitParts** 中
2. 每个文件的图标应该是脚本图标（看起来像一张纸）
3. 运行游戏，打开背包 → 主角装扮，应该能看到列表

### 常见问题

**Q: Create 菜单中没有"绒毛几何物语"选项**
A: 检查 Assets/Scripts/Data/OutfitPartData.cs 文件是否存在，且第7行是：
```csharp
[CreateAssetMenu(fileName = "NewOutfitPart", menuName = "绒毛几何物语/装扮部件数据")]
```

**Q: 创建了但游戏中看不到**
A: 1. 确保文件在 Assets/Resources/OutfitParts 下
   2. 检查文件名和 Part Id 是否一致
   3. 确保 Is Unlocked By Default 已勾选
   4. 重启 Unity 试试

**Q: 部件列表为空**
A: 1. 确保场景中有 OutfitManager 游戏对象
   2. 确保 OutfitManager 脚本已启用
   3. 查看 Console 是否有错误信息
