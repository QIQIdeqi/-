## 核心定位

### 目标用户
- **主要受众：** 18-35岁女性玩家
- **次要受众：** 喜欢治愈系、收集类游戏的泛用户
- **用户画像：** 追求轻松体验、喜爱可爱画风、享受收集和装饰乐趣

### 一句话描述
玩家扮演编织精灵，在温暖的羊毛毡世界中探索收集，用编织魔法安抚调皮的手工元素，打造属于自己的治愈小屋。

### 核心体验转变
| 维度 | 旧定位（几何战士） | 新定位（绒毛几何物语） |
|------|------------------|---------------------|
| 核心乐趣 | 割草爽快感 | 收集满足感 |
| 美术风格 | 霓虹赛博 | 羊毛毡手作 |
| 情绪基调 | 紧张刺激 | 温暖治愈 |
| 玩法重心 | 战斗 > 收集 | 收集 > 战斗 |
| 互动反馈 | 消灭/击杀 | 安抚/整理/修复 |
| 成长系统 | 武器升级 | 皮肤/家园/图鉴 |

### 游戏循环
```
开始探索 → 发现调皮元素 → 用编织魔法安抚 → 收集材料 → 
解锁新皮肤/家具/装饰 → 装扮精灵/布置家园 → 获得满足感 → 继续探索
```

## 世界观设定

### 背景故事
你是一个诞生于「永恒编织篮」的编织精灵。某天，一股「混乱毛线」的能量席卷了原本宁静的手工世界，使得毛线团四处滚动、纽扣错位、针脚脱落。你的使命是漫步于这个巨大的羊毛毡箱庭，用编织魔法安抚调皮的元素，收集散落的珍贵材料，让世界恢复和谐与可爱。

### 世界构成
- **永恒编织篮：** 玩家的家园/小屋，可自由布置
- **毛线森林：** 由彩色毛线缠绕的树木、毛毡叶片构成的森林区域
- **纽扣平原：** 米色帆布地面，散落着彩色纽扣路径
- **碎布海湾：** 蓝色系碎布拼接的海洋，棉花云朵漂浮
- **刺绣山谷：** 满是精美刺绣图案的开阔地带

## 核心机制

### 1. 角色系统（编织精灵）

#### 基础形态
- **核心造型：** 圆润可爱的毛球/方盒玩偶，由羊毛毡制成
- **移动方式：** 漂浮移动，带有轻微的弹性动画
- **自动安抚：** 编织工具自动发射（无需点击），轨迹是柔软的毛线或闪烁的缝针光

#### 皮肤系统（核心收集要素）
玩家可以通过收集材料，为精灵编织数十种不同「皮肤套装」：

| 皮肤类型 | 描述 | 解锁条件 |
|---------|------|---------|
| 小熊套装 | 棕色毛毡小熊外观，有圆圆的耳朵 | 初始解锁 |
| 小猫套装 | 软萌猫咪造型，带铃铛项圈 | 收集10个毛线球 |
| 小兔套装 | 长耳朵兔子，粉色内耳 | 收集5个胡萝卜刺绣 |
| 公主裙套装 | 粉色纱裙+小皇冠 | 完成「针线城堡」区域 |
| 云朵套装 | 棉花糖般的蓬松外观 | 收集20朵棉花云 |
| 复古套装 | 粗花呢材质+贝雷帽 | 收集10个复古顶针 |

#### 陪伴可视化（能力外显）
- **初级：** 精灵身边漂浮一个小毛线球宠物
- **中级：** 增加一个毛线钩织的小小守护天使
- **高级：** 一条会随风飘动的编织围巾 + 发光效果

### 2. 互动对象设计（调皮元素 → 需要安抚的朋友）

#### 设计哲学
不再是「敌人」，而是世界中需要被「安抚」、「整理」或「修复」的调皮元素。设计必须极致可爱，符合「即使被'打败'也让人会心一笑」的原则。

#### 互动类型

**🧶 毛线团怪（基础互动对象）**
- **外观：** 一团颜色杂乱、拖着许多线头的毛线球
- **行为：** 四处滚动，偶尔缠住自己
- **互动结果：** 被「梳理」成一个颜色纯净、整齐圆润的完美毛线球
- **掉落：** 毛线球材料 + 毛线团怪图鉴
- **反馈特效：** 毛线被梳理整齐时的顺滑动画 + 温暖星光

**🧸 脱线小兵（中等互动对象）**
- **外观：** 简单几何体构成的小玩偶，身体上有明显的脱线处
- **行为：** 摇摇晃晃地走动，偶尔坐下「休息」
- **互动结果：** 身上的线头被自动缝好，变成一个友好NPC
- **掉落：** 缝纫针材料 + 脱线小兵图鉴
- **反馈特效：** 缝线「嗖」地缝合的动画 + 小玩偶开心的跳跃

**🎀 纽扣眼魔偶（精英互动对象）**
- **外观：** 可爱的布偶，但纽扣眼睛歪了，显得有点「丧」
- **行为：** 慢慢踱步，歪头看你
- **互动结果：** 纽扣「啪嗒」一声归位，魔偶开心跳一跳
- **掉落：** 特殊纽扣 + 纽扣眼魔偶图鉴 + 小家具图纸
- **反馈特效：** 纽扣旋转归位 + 小星星闪烁

**🌸 混乱花束（区域Boss）**
- **外观：** 一大束颜色冲突、枝条杂乱的毛毡花
- **行为：** 花枝摇摆，偶尔掉落杂色花瓣
- **互动结果：** 被整理成和谐的渐变色花束
- **掉落：** 稀有花种 + 大型家具图纸 + 混乱花束图鉴
- **反馈特效：** 花朵颜色渐变和谐 + 花瓣雨动画

### 3. 编织工具系统（原武器系统）

#### 基础工具
| 工具 | 描述 | 效果 |
|-----|------|------|
| 毛线针 | 发射细长的毛线轨迹 | 单体安抚 |
| 顶针泡泡 | 发射柔软的泡泡 | 范围安抚 |
| 毛线球滚轮 | 滚动的大毛线球 | 路径安抚 |

#### 升级路线（3选1）
**毛线针路线：**
- 细毛线针 → 粗毛线针（范围增大） → 彩虹毛线针（多色轨迹）

**顶针泡泡路线：**
- 小泡泡 → 大泡泡（范围增大） → 连环泡泡（分裂效果）

**毛线球滚轮路线：**
- 小毛线球 → 大毛线球（范围增大） → 弹力毛线球（弹射效果）

#### 特殊工具（进化）
满级工具 + 收集核心道具 = 特殊编织工具：
- **编织光环：** 精灵周围持续产生编织波动
- **缝纫闪电：** 连锁缝合多个目标
- **彩虹编织：** 全屏彩色编织特效

### 4. 家园系统（核心玩法承载）

#### 手作小屋
玩家的个人空间，可以自由布置：
- **墙面：** 可选择不同材质（毛毡墙、布艺墙、刺绣墙）
- **地板：** 毛线地毯、纽扣拼接地砖、软木地板
- **家具：** 羊毛毡沙发、纽扣相框、编织花篮、刺绣挂画
- **装饰：** 收集的小摆件、图鉴奖杯、季节性装饰

#### 布置玩法
- 拖拽摆放家具和装饰
- 旋转、缩放部分物品
- 保存多个布局方案
- 拜访好友的小屋（微信好友）

### 5. 收集系统

#### 材料收集
- **毛线球：** 基础材料，用于制作皮肤和家具
- **纽扣：** 装饰材料，用于解锁特殊皮肤
- **顶针：** 稀有材料，用于高级家具
- **刺绣线：** 特殊材料，用于图鉴升级
- **棉花云：** 家园装饰专用材料

#### 图鉴系统
- **调皮元素图鉴：** 记录所有安抚过的「朋友」
- **皮肤图鉴：** 展示已收集的皮肤
- **家具图鉴：** 展示已制作的家具
- **材料图鉴：** 记录收集过的所有材料

#### 成就系统
- 「初次相遇」：安抚第一个调皮元素
- 「编织大师」：收集所有基础皮肤
- 「家园设计师」：布置手作小屋
- 「图鉴收集家」：完成区域图鉴

## 美术风格：羊毛毡手作

### 视觉方向
- **风格：** 羊毛毡手工艺风
- **材质：** 模拟羊毛毡、毛线、不织布、棉絮、纽扣、拉链等真实手工材料
- **色调：** 马卡龙色系，低饱和度 pastel colors
- **氛围：** 温暖、治愈、可爱、手作感

### 配色方案
| 元素 | 主色 | 辅色 |
|------|------|------|
| 玩家精灵 | 奶油黄 #FFF8DC | 浅粉 #FFD1DC |
| 毛线团怪 | 暖棕 #D2691E | 米白 #FFFAF0 |
| 脱线小兵 | 浅蓝 #ADD8E6 | 灰色 #D3D3D3 |
| 纽扣眼魔偶 | 粉色 #FFB6C1 | 深蓝 #191970 |
| 森林区域 | 薄荷绿 #98FF98 | 草绿 #90EE90 |
| 平原区域 | 米色 #F5F5DC | 彩色纽扣 |
| 海湾区域 | 天蓝 #87CEEB | 白色棉花 |

### 特效风格
- **安抚命中：** 爆出一团蓬松的彩色绒毛或七彩线头星光
- **收集物品：** 物品轻微毛茸茸弹跳 + 温暖星光粒子
- **升级/进化：** 毛线编织完成的动画 + 柔和光晕
- **治愈/修复：** 像小布料被迅速缝补好的动画

### UI设计
- **按钮：** 软木塞质感、缝线边框
- **进度条：** 正在被编织的毛线，逐渐变长
- **图标：** 毛线刺绣风格
- **弹窗：** 亚麻布背景 + 标签样式边框

## 变现设计（女性向适配）

### 广告点
| 位置 | 类型 | 奖励 | 设计要点 |
|------|------|------|---------|
| 能量耗尽 | 激励视频 | 继续探索 | 用「休息一会儿」替代「死亡」概念 |
| 材料双倍 | 激励视频 | 本次收集x2 | 突出「更多材料=更快装扮家园」 |
| 每日礼物 | 激励视频 | 限定装饰品 | 强调「限定」「稀有」 |
| 解锁皮肤 | 激励视频 | 预览/试穿皮肤 | 满足「爱美」心理 |

### 内购设计
- **去广告礼包：** 一次购买，永久无广告
- **材料月卡：** 每日领取材料礼包
- **限定皮肤包：** 季节/节日限定皮肤
- **家园扩展包：** 解锁更多房间/区域
- **好友礼物：** 给微信好友送材料/装饰品
