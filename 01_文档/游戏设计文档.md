## 核心玩法

### 一句话描述
玩家控制几何形状战士，在霓虹几何世界中自动射击，消灭来袭的敌人几何体，收集能量升级武器。

### 游戏循环
```
开始游戏 → 击杀几何敌人 → 收集能量核心 → 升级选武器/技能 → 变强 → 面对更复杂几何敌人 → 循环
```

---

## 核心机制

### 1. 角色系统
- 自动攻击（无需点击）
- 移动控制（虚拟摇杆/触屏拖动）
- 生命值系统（护盾+血条）

### 2. 敌人系统
- 波次生成
- **敌人类型：**
  - ▲ 三角形 - 近战冲锋
  - ◆ 菱形 - 远程射击
  - ■ 正方形 - 坦克型，血厚
  - ★ 五角星 - 精英怪，组合攻击
  - ● 圆形Boss - 阶段变化
- 掉落能量核心（经验）

### 3. 武器/技能系统
- **开局武器：** 基础激光
- **升级时3选1：**
  - 双激光 → 散射激光 → 激光风暴
  - 追踪导弹 → 分裂导弹 → 核弹
  - 旋转护盾 → 锯齿护盾 → 毁灭光环
  - 电磁脉冲 → 连锁闪电 → 雷霆万钧
- **进化条件：** 武器满级 + 拾取核心道具

### 4. 成长系统
- 局内：升级选武器/技能
- 局外：能量币强化基础属性（移速、攻击、血量）

## 美术风格：霓虹几何

### 视觉方向
- **风格：** 霓虹赛博几何风
- **色调：** 深色背景 + 高饱和度霓虹色（青、粉、紫、绿）
- **元素：** 发光边缘、粒子拖尾、几何破碎效果
- **氛围：** 科技感、节奏感、打击感

### 配色方案
| 元素 | 主色 | 发光色 |
|------|------|--------|
| 玩家 | 青色 #00FFFF | 亮青 #80FFFF |
| 敌人-近战 | 红色 #FF0040 | 亮红 #FF80A0 |
| 敌人-远程 | 绿色 #00FF40 | 亮绿 #80FFA0 |
| 敌人-坦克 | 橙色 #FF8000 | 亮橙 #FFC080 |
| 能量核心 | 金色 #FFD700 | 亮金 #FFEE80 |
| 背景 | 深紫 #0A0A1A | 网格线 #1A1A3A |

### 特效风格
- 击中：几何碎片飞溅 + 霓虹闪烁
- 升级：全屏脉冲波纹 + 粒子爆发
- 死亡：几何体解体动画
