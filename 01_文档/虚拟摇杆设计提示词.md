# 虚拟摇杆设计提示词 - 绒毛几何物语

## 设计说明

### 组成部分
- **JoystickBackground（底座）**: 固定不动的圆形底座
- **JoystickHandle（摇杆柄）**: 跟随手指移动的中心控制球

### 尺寸建议
- **Background**: 200x200 像素（外圈直径）
- **Handle**: 80x80 像素（内圈直径，可移动范围约60像素半径）

### 风格统一
- 羊毛毡材质
- 马卡龙色系
- 手工缝制细节
- 温暖治愈感

---

## Unity UI 设置建议

### 摇杆组件配置
```csharp
// Joystick组件结构
- Joystick (空物体，中心锚点)
  - Background (Image，底座Sprite)
    - Handle (Image，摇杆柄Sprite，Background的子物体)

// 尺寸设置
Background RectTransform: 
  - Size: 200x200
  - Anchors: Center

Handle RectTransform:
  - Size: 80x80
  - Anchors: Center
  - Position: (0, 0) 默认居中

// 移动范围
Max Movement Radius: 60 pixels (Handle可移动的最大距离)
```

### Sprite导入设置
1. **Texture Type**: Sprite (2D and UI)
2. **Sprite Mode**: Single
3. **Pixels Per Unit**: 100
4. **Generate Mip Maps**: Off

### 颜色配置建议
| 元素 | 推荐颜色 | 说明 |
|-----|---------|------|
| Background | #FFF8DC (奶油) | 温暖中性底 |
| Handle | #FFB6C1 (粉) 或 #98FF98 (绿) | 突出可交互 |
| 缝线/装饰 | #D2691E (巧克力) | 对比细节 |

### 交互反馈
```csharp
// 可选：按下时的视觉反馈
- Background: Scale 0.95, 轻微变暗
- Handle: Scale 1.1, 突出显示
- 两者都添加轻微弹性动画
```

## 配套设计：禁用/非激活状态

### JoystickBackground (Disabled)
```
A disabled/inactive version of the felt joystick base.

Changes:
- Grayscale or desaturated colors
- Reduced opacity (70%)
- No interactive highlights
- Shadow less pronounced

Style: Same texture, muted appearance

Parameters: --ar 1:1 --style raw --v 6.0 --no background
```

### JoystickHandle (Disabled)
```
A disabled/inactive version of the felt joystick handle.

Changes:
- Grayed out color
- Locked in center position

Style: Same shape, inactive state
```

## 方案对比总结

| 方案 | Background风格 | Handle风格 | 主题契合度 | 复杂度 |
|-----|---------------|-----------|-----------|-------|
| 毛线团 | 缠绕毛线底座 | 毛球 | ⭐⭐⭐⭐⭐ | 高 |
| 大纽扣 | 巨型纽扣 | 小纽扣 | ⭐⭐⭐⭐ | 中 |
| 编织垫 | 钩针编织 | 绒球 | ⭐⭐⭐⭐⭐ | 高 |

**推荐：方案1 毛线团风格**
- 最符合绒毛几何物语主题
- 底座像大毛线团，摇杆像小毛球
- 材质和纹理统一
