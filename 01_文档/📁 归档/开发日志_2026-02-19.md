## 代码开发

### 1. 创建武器系统核心代码

**WeaponBase.cs** - 武器基类
```csharp
namespace GeometryWarrior
{
    public abstract class WeaponBase : MonoBehaviour
    {
        protected string weaponName = "Weapon";
        protected int currentLevel = 1;
        protected int maxLevel = 5;
        
        public abstract void PerformAttack();
        public virtual void Upgrade() { currentLevel++; }
    }
}
```

**LaserWeapon.cs** - 激光武器
- 发射穿透性激光束
- 攻击间隔：0.8秒
- 伤害：15 + (等级-1)*5
- 5级效果：同时发射3道散射激光

**MissileWeapon.cs** - 追踪导弹武器
- 发射自动追踪敌人的导弹
- 攻击间隔：1.2秒
- 伤害：25 + (等级-1)*5
- 升级效果：增加导弹数量、爆炸范围、速度

**MissileProjectile.cs** - 导弹投射物
- 自动追踪最近敌人
- 击中敌人或障碍物时爆炸
- 范围伤害（随距离衰减）

**WeaponManager.cs** - 武器管理器
- 单例模式

## 关键修复记录

### 修复1：Player 事件绑定问题
**问题**：Player 是游戏开始后才生成，UpgradeManager 找不到 Player
**解决**：添加 GameManager.OnPlayerSpawned 事件，等 Player 生成后再订阅升级事件

```csharp
// GameManager.cs
public System.Action<PlayerController> OnPlayerSpawned;

private void SpawnPlayer()
{
    // ... 实例化 Player ...
    OnPlayerSpawned?.Invoke(Player);
}

// UpgradeManager.cs
private void Start()
{
    GameManager.Instance.OnPlayerSpawned += OnPlayerSpawned;
}

private void OnPlayerSpawned(PlayerController spawnedPlayer)
{
    player = spawnedPlayer;
    player.OnLevelUp += OnPlayerLevelUp;
}
```

### 修复2：中文显示问题
**问题**：TextMeshPro 字体缺少中文字符，显示为方框
**解决**：改用 Unity 默认 Text 组件，支持中文显示

```csharp
// UpgradeOptionUI.cs
[SerializeField] private Text nameTextLegacy;  // Unity default Text
[SerializeField] private Text descriptionTextLegacy;
```

### 修复3：激光武器警告
**问题**：maxBounces 字段未使用
**解决**：改为 laserCount，实现5级多道激光效果

```csharp
// 修改前
if (other.CompareTag("Obstacle")) // 报错！

// 修改后

    other.gameObject.layer == LayerMask.NameToLayer("Obstacle") ||
    other.name.Contains("Obstacle") ||
    other.name.Contains("Wall"))
```
