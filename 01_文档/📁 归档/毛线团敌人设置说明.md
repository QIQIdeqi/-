# 毛线团敌人设置说明

## 文件说明

### 1. YarnBallDropEffect.cs
**路径：** `Assets/Scripts/Effects/YarnBallDropEffect.cs`

毛线团掉落效果组件，处理：
- 向上飞的初始速度
- 重力下落
- 旋转动画
- 地面反弹
- 自动淡出消失

### 2. Enemy_YarnBall.cs
**路径：** `Assets/Scripts/Enemy/Enemy_YarnBall.cs`

毛线团敌人脚本，继承自 EnemyBase，添加：
- 死亡时生成掉落物
- 存活时的摇摆动画

---

## Unity设置步骤

### 第一步：创建毛线团掉落预制体

1. **创建空物体**
   - Hierarchy → 右键 → Create Empty
   - 命名为 `YarnBall_Drop`

2. **添加SpriteRenderer**
   - 添加组件：Sprite Renderer
   - 设置 Sprite：导入的毛线团贴图（整齐状态的毛线球）
   - Sorting Layer：设置为与敌人相同的层（如 `Enemies`）

3. **添加碰撞器（可选，用于玩家拾取）**
   - 添加组件：Circle Collider 2D
   - Is Trigger：✅ 勾选
   - Radius：根据贴图大小调整

4. **添加脚本**
   - 添加组件：`YarnBallDropEffect` 脚本
   - 配置参数：
     - Pop Up Force：5（向上飞的力度）
     - Gravity：15（下落速度）
     - Ground Bounce：0.3（地面反弹）
     - Ground Y：-4（地面高度，根据你的场景调整）
     - Rotation Speed：360（旋转速度）
     - Disappear Delay：2（落地后停留时间）

5. **保存预制体**
   - 拖到 Project 窗口的 `Assets/Prefabs/Effects/` 文件夹

---

### 第二步：创建毛线团敌人预制体

1. **复制现有敌人预制体**
   - 找到现有的敌人预制体（如 `Enemy_Triangle`）
   - 复制一份，命名为 `Enemy_YarnBall`

2. **修改Sprite**
   - 将 SpriteRenderer 的 Sprite 改为毛线团怪（混乱状态的毛线球）

3. **替换脚本**
   - 移除原有的敌人脚本（如果有）
   - 添加 `Enemy_YarnBall` 脚本

4. **配置参数**
   - **Base Stats**：设置血量、速度等
   - **Yarn Ball Drop**：
     - Yarn Ball Drop Prefab：拖拽刚才创建的 `YarnBall_Drop` 预制体
     - Yarn Ball Sprite：拖拽整齐状态的毛线球贴图
     - Drop Y Offset：0（生成位置偏移）
   - **Yarn Ball Visuals**：
     - Body Renderer：SpriteRenderer 引用（自动或手动）
     - Wobble Speed：3（摇摆速度）
     - Wobble Amount：5（摇摆幅度）

5. **添加碰撞器**
   - Circle Collider 2D（如果还没有）
   - Rigidbody 2D（如果还没有，设置为 Dynamic，Gravity Scale = 0）

6. **设置标签和层**
   - Tag：`Enemy`
   - Layer：`Enemy`

7. **保存预制体**

---

### 第三步：添加到敌人生成系统

1. **找到敌人生成器**
   - 通常是 `EnemySpawner` 或 `WaveManager`

2. **添加新敌人类型**
   - 在敌人列表中添加 `Enemy_YarnBall`
   - 设置生成权重和波次

## 参数调整建议

### YarnBallDropEffect 参数

| 参数 | 默认值 | 说明 | 调整建议 |
|-----|-------|------|---------|
| Pop Up Force | 5 | 向上飞的力度 | 越大飞得越高 |
| Gravity | 15 | 重力 | 越大落得越快 |
| Ground Bounce | 0.3 | 反弹系数 | 0-1之间，越大弹得越高 |
| Rotation Speed | 360 | 旋转速度 | 度/秒，越大转得越快 |
| Disappear Delay | 2 | 停留时间 | 落地后多久消失 |
| Fade Out Duration | 0.5 | 淡出时间 | 消失时的渐变时间 |

### Enemy_YarnBall 参数

| 参数 | 默认值 | 说明 | 调整建议 |
|-----|-------|------|---------|
| Wobble Speed | 3 | 摇摆速度 | 越大摇摆越快 |
| Wobble Amount | 5 | 摇摆角度 | 度，越大倾斜越明显 |

## 常见问题

### Q: 毛线团飞得太高/太低
**A:** 调整 `YarnBallDropEffect` 的 `Pop Up Force` 参数

### Q: 落地后弹得太高
**A:** 调整 `Ground Bounce` 参数，设为 0 则不反弹

### Q: 旋转太快/太慢
**A:** 调整 `Rotation Speed` 参数

### Q: 毛线团在错误的高度消失
**A:** 确保 `Ground Y` 参数与你的场景地面高度匹配

## 扩展功能

### 添加拾取功能
在 `YarnBallDropEffect.OnPickup()` 中添加：
```csharp
// 增加材料
Inventory.Instance.AddItem(ItemType.YarnBall, 1);

// 播放音效
AudioManager.Instance.PlaySFX("pickup_yarn");

// 显示 floating text
UIManager.Instance.ShowFloatingText("+1 毛线", transform.position);
```

### 添加物理碰撞
如果需要毛线团之间可以碰撞，添加 Rigidbody2D：
- Body Type：Dynamic
- Gravity Scale：0（由脚本控制重力）
- Collision Detection：Continuous
