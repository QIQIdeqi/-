## 解决方案备忘

### 微信小游戏导出完整配置步骤

**步骤1：安装 WebGL 模块**
1. Unity Hub → 安装 → 找到当前 Unity 版本
2. 点击齿轮 → 添加模块
3. 勾选 **WebGL Build Support**
4. 安装后重启 Unity

**步骤2：Player Settings 配置**
路径：`Edit` → `Project Settings` → `Player` → `Other Settings`

| 配置项 | 设置值 |
|--------|--------|
| **Color Space** | `Gamma` |
| **Graphics APIs** | 添加 `WebGL 2` |

**步骤3：微信插件配置**
1. 导入 `minigame.unitypackage`
2. 菜单 `微信小游戏` → `转换小游戏`
3. 填写 AppID 和导出路径
4. 点击「生成并转换」
