# 几何战士 - Unity 组件制作步骤 (2026-02-17)

## 一、创建文件夹结构

在 `Assets/` 下创建：
```
Assets/
├── Scripts/
│   ├── Player/      # 玩家相关脚本
│   ├── Enemy/       # 敌人相关脚本
│   ├── Manager/     # 管理器脚本
│   └── UI/          # UI脚本
├── Prefabs/         # 预制体
├── Sprites/         # 图片资源
└── Scenes/          # 场景文件
```

---

## 二、玩家预制体 (Player Prefab)

### 1. 创建玩家对象
1. Hierarchy 右键 → `Create Empty` → 命名为 `Player`
2. 添加组件：
   - `Sprite Renderer`
     - Sprite: 选择圆形或创建 `Circle` (右键 → 2D → Sprites → Circle)
     - Color: 青色 (0, 255, 255)
   - `Rigidbody 2D`
     - Gravity Scale: 0
     - Freeze Rotation: ✓
   - `Circle Collider 2D`
   - `PlayerController` 脚本

### 2. 创建子弹预制体
1. Hierarchy 右键 → `Create Empty` → 命名为 `Projectile`
2. 添加组件：
   - `Sprite Renderer`
     - Sprite: 小圆点
     - Color: 黄色
   - `Rigidbody 2D`
     - Gravity Scale: 0
   - `Circle Collider 2D`
     - Is Trigger: ✓
   - `Projectile` 脚本
3. 拖到 `Prefabs/` 文件夹保存

### 3. 配置 PlayerController
| 参数 | 值 |
|------|-----|
| Move Speed | 5 |
| Max Health | 100 |
| Attack Range | 8 |
| Attack Cooldown | 0.5 |
| Projectile Prefab | 拖入 Projectile 预制体 |
| Attack Damage | 20 |

---

## 三、敌人预制体 (Enemy Prefab)

### 1. 创建敌人对象
1. Hierarchy 右键 → `Create Empty` → 命名为 `Enemy`
2. 添加组件：
   - `Sprite Renderer`
     - Sprite: 三角形 (右键 → 2D → Sprites → Triangle)
     - Color: 红色
   - `Rigidbody 2D`
     - Gravity Scale: 0
     - Freeze Rotation: ✓
   - `Polygon Collider 2D` (或 Box Collider 2D)
   - `EnemyBase` 脚本

### 2. 配置 EnemyBase
| 参数 | 值 |
|------|-----|
| Max Health | 30 |
| Attack Damage | 10 |
| Move Speed | 2 |
| Exp Value | 10 |
| Exp Orb Prefab | 拖入 ExpOrb 预制体 |

### 3. 保存为预制体
- 拖到 `Prefabs/` 文件夹

---

## 四、经验球预制体 (ExpOrb Prefab)

### 创建经验球
1. Hierarchy 右键 → `Create Empty` → 命名为 `ExpOrb`
2. 添加组件：
   - `Sprite Renderer`
     - Sprite: 圆形
     - Color: 黄色
   - `Circle Collider 2D`
     - Is Trigger: ✓
   - `ExpOrb` 脚本
3. 配置参数：
   - Exp Value: 10
   - Attract Range: 3
   - Attract Speed: 8
   - Lifetime: 10
4. 拖到 `Prefabs/` 保存

---

## 五、场景设置

### 1. 创建 GameScene
1. `File` → `New Scene`
2. `File` → `Save Scene As` → `Scenes/GameScene`

### 2. 添加相机
1. 选中 Main Camera
2. Camera 组件设置：
   - Projection: Orthographic
   - Size: 10
3. 添加 `CameraFollow` 脚本
   - Target: 拖入 Player

### 3. 添加 GameManager
1. Hierarchy 右键 → `Create Empty` → 命名为 `GameManager`
2. 添加 `GameManager` 脚本
3. 配置参数：
   - Player Prefab: 拖入 Player 预制体
   - Main Menu Panel: 稍后配置
   - Game Over Panel: 稍后配置

### 4. 添加 EnemySpawner
1. Hierarchy 右键 → `Create Empty` → 命名为 `EnemySpawner`
2. 添加 `EnemySpawner` 脚本
3. 配置参数：
   - Enemy Prefab: 拖入 Enemy 预制体
   - Spawn Interval: 1
   - Max Enemies: 30

---

## 六、UI 制作

### 1. 创建 Canvas
1. Hierarchy 右键 → `UI` → `Canvas`
2. Canvas Scaler 设置：
   - UI Scale Mode: Scale With Screen Size
   - Reference Resolution: 1920 x 1080

### 2. 创建 HUD (游戏内界面)
1. 在 Canvas 下创建空物体 → 命名为 `HUD`
2. 添加 `GameHUD` 脚本
3. 创建以下 UI 元素：

**血条**
- 右键 → UI → Slider → 命名为 `HealthBar`
- 删除 Handle Slide Area
- Fill Area → Fill 颜色改为红色

**经验条**
- 复制 HealthBar → 命名为 `ExpBar`
- Fill 颜色改为绿色

**分数文本**
- 右键 → UI → Text - TextMeshPro → 命名为 `ScoreText`
- 内容: "Score: 0"

**时间文本**
- 右键 → UI → Text - TextMeshPro → 命名为 `TimeText`
- 内容: "00:00"

**等级文本**
- 右键 → UI → Text - TextMeshPro → 命名为 `LevelText`
- 内容: "Lv.1"

4. 将以上元素拖到 GameHUD 脚本的对应字段

### 3. 创建主菜单面板
1. 在 Canvas 下创建空物体 → 命名为 `MainMenuPanel`
2. 添加 Image 组件作为背景
3. 添加 `MainMenuPanel` 脚本
4. 添加按钮：
   - 开始游戏按钮 (StartGameButton)
   - 设置按钮 (SettingsButton)
   - 退出按钮 (QuitButton)
5. 绑定按钮点击事件

### 4. 创建游戏结束面板
1. 在 Canvas 下创建空物体 → 命名为 `GameOverPanel`
2. 添加 Image 组件作为背景
3. 添加 `GameOverPanel` 脚本
4. 添加元素：
   - 分数显示文本
   - 存活时间文本
   - 获得能量币文本
   - 最高分文本
   - 复活按钮
   - 重新开始按钮
   - 返回菜单按钮

## 七、最终配置

### GameManager 引用绑定
| 字段 | 拖入对象 |
|------|---------|
| Player Prefab | Player 预制体 |
| Spawn Point | 场景中的空物体或 Player 本身 |
| Main Menu Panel | MainMenuPanel 对象 |
| Game Over Panel | GameOverPanel 对象 |
| HUD Canvas | HUD 对象 |

### Enemy 预制体引用
- 在 Enemy 预制体的 EnemyBase 脚本中：
  - Exp Orb Prefab: 拖入 ExpOrb 预制体

### Player 预制体引用
- 在 Player 预制体的 PlayerController 脚本中：
  - Projectile Prefab: 拖入 Projectile 预制体
