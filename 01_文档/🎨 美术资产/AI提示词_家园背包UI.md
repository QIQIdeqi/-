## 🔄 迭代优化建议

### 如果生成结果太写实：
- 添加 `cartoon style, flat illustration`
- 添加 `no realistic details, simplified shapes`
- 提高 `--s` 值增加风格化

### 如果纹理不够明显：
- 添加 `visible fabric texture, clear wool fibers`
- 添加 `close-up texture detail`

### 如果色彩不对：
- 使用 `hex color code` 直接指定
- 添加 `color accurate, exact color match`
- 使用 `--no` 排除错误颜色

### 如果需要透明背景：
- Midjourney: 生成后手动抠图，或使用 `transparent background`（效果不稳定）
- 建议：生成白色背景版本，后期用 Photoshop 或 remove.bg 处理

## 💡 实用技巧

1. **一致性控制**：使用相同的 `--seed` 值可以生成风格相似的系列
2. **批量生成**：将类似元素组合在一起生成，如 `3 cute felted wool buttons in different colors`
3. **参考图**：上传参考图 + `--iw 0.5` 保持风格一致
4. **局部重绘**：使用 Vary Region 功能调整不满意的部分
