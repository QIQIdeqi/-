## 批量生成建议

### 使用 Midjourney
1. 将上述英文提示词逐条输入
2. 使用 `/imagine` 命令
3. 选择满意的图片进行 Upscale
4. 下载后使用图像编辑软件去除背景

### 使用 DALL-E 3 / GPT Image
1. 使用 OpenAI 平台
2. 建议参数：1024x1024, high quality
3. 添加 "transparent background" 要求
4. 可能需要后期手动去背

### 使用 Leonardo.ai 或其他工具
- 提示词结构相同
- 注意选择适合游戏素材的模型
- 可能需要调整负面提示词

## 质量检查清单

生成后检查每个素材是否满足：
- [ ] 是否为正俯视视角？
- [ ] 是否有羊毛毡/毛绒质感？
- [ ] 颜色是否为马卡龙/ pastel 色系？
- [ ] 是否有可见的手缝线细节？
- [ ] 光影是否柔和温暖？
- [ ] 是否有「可爱」「治愈」的感觉？
- [ ] 是否避免了尖锐边缘？
- [ ] 是否避免了霓虹/赛博元素？
