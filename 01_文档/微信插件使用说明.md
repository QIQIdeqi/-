# 微信小游戏插件使用说明

## 官方资源

**插件仓库：** https://gitee.com/wechat-minigame/minigame-unity-webgl-transform

**官方文档：** https://developers.weixin.qq.com/minigame/dev/guide/framework/unity-plugin.html

---

## 下载插件

### 方式1：直接下载
1. 访问 https://gitee.com/wechat-minigame/minigame-unity-webgl-transform
2. 点击「下载」→「下载ZIP」
3. 解压后将 `minigame.unitypackage` 导入Unity

### 方式2：Git克隆
```bash
cd D:\03_独立游戏\自己\微信小游戏\03_工具\微信配置
git clone https://gitee.com/wechat-minigame/minigame-unity-webgl-transform.git
```

---

## 导入Unity

1. 打开Unity项目
2. `Assets` → `Import Package` → `Custom Package...`
3. 选择下载的 `minigame.unitypackage`
4. 点击 Import

---

## 配置导出

1. Unity菜单：`微信小游戏` → `转换小游戏`
2. 填写配置：
   - **AppID**: `wxa6101eec77c5541d`
   - **游戏名称**: `几何战士`
   - **导出路径**: `D:\03_独立游戏\自己\微信小游戏\01_项目\项目01_肉鸽割草\05_导出\微信小游戏`
   - **首包资源加载方式**: 包内加载（建议）
3. 点击「生成并转换」

---

## 测试运行

1. 打开「微信开发者工具」
2. 导入项目 → 选择导出目录
3. 填入 AppID：`wxa6101eec77c5541d`
4. 点击「真机调试」或「预览」

---

## 常见问题

### 导出失败
- 检查Unity版本是否兼容（2022.3 LTS）
- 确保没有编译错误
- 检查导出路径是否有空格或中文

### 运行时黑屏
- 检查首包资源加载方式设置
- 确认资源没有超出大小限制

### 性能问题
- 使用对象池管理敌人
- 控制同屏粒子数量
- 开启IL2CPP编译优化
