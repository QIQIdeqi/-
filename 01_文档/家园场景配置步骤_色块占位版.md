# 家园场景配置指南 - 色块占位版

## 📋 配置概述

在AI美术资产生成完成前，先用色块占位快速搭建场景框架，后续直接替换Sprite即可。

---

## 🎯 Unity场景配置步骤

### 1. 创建基础场景结构

在 `HomeScene` 中创建以下物体：

```
HomeScene
├── 【管理器】
│   └── HomeManager (空物体)
├── 【环境】
│   ├── FloorGrid (空物体，地板网格)
│   ├── LeftWall (Sprite，左墙)
│   └── RightWall (Sprite，右墙)
├── 【交互物体】
│   ├── HomeNPC (Sprite，NPC)
│   ├── HomeDoor (Sprite，门)
│   └── Decorations (空父物体)
│       ├── Deco_地毯 (Sprite)
│       ├── Deco_沙发 (Sprite)
│       └── Deco_盆栽 (Sprite)
├── 【玩家】
│   └── PlayerSpawnPoint (空物体)
└── 【UI】Canvas
    ├── OutfitPanel (Panel，默认隐藏)
    ├── InteractionHint (Text，提示"按E对话")
    └── DoorArrow (Image，离开箭头)
```

### 3. HomeManager 配置

选中 `HomeManager` 物体，Inspector中设置：

```
【场景设置】
- Main Menu Scene Name: MainScene

【家园边界】
- Home Bounds:
  - Center: (0, 0, 0)
  - Size: (8, 6, 0)

【玩家设置】
- Player Prefab: 拖入玩家预制体
- Player Spawn Point: 拖入 PlayerSpawnPoint

【装饰物】
- Decorations: 拖入所有 Deco_xxx 物体

【NPC和门】
- Home NPC: 拖入 HomeNPC
- Home Door: 拖入 HomeDoor
```

---

### 4. HomeNPC 配置

选中 `HomeNPC` 物体：

```
【交互设置】
- Interaction Radius: 1.5
- Interaction Key: E
- Interaction Hint: 拖入 Canvas/InteractionHint
- Outfit Panel: 拖入 Canvas/OutfitPanel

【外观】
- SpriteRenderer: Circle (临时)
- Color: #FF69B4
```

---

### 5. HomeDoor 配置

选中 `HomeDoor` 物体：

```
【触发设置】
- Trigger Radius: 1
- Arrow Indicator: 拖入 Canvas/DoorArrow
- Leave Hint: 可留空或使用InteractionHint
- Leave Button: 可留空（接触自动触发）

【外观】
- SpriteRenderer: Square
- Color: #DDA0DD
```

---

### 6. OutfitPanel UI 配置

在 Canvas 下创建 `OutfitPanel`：

#### 层级结构：
```
OutfitPanel (Panel)
├── Title (TextMeshPro): "装扮小屋"
├── CloseButton (Button): X
├── CategoryTabs (Horizontal Layout Group)
│   ├── Tab_全部 (Button)
│   ├── Tab_蝴蝶结 (Button)
│   ├── Tab_帽子 (Button)
│   └── Tab_眼镜 (Button)
├── PartsGrid (Grid Layout Group)
│   └── PartItemPrefab (Button，预制体)
├── PreviewArea (Panel)
│   └── PreviewPlayer (Sprite: 玩家预览图)
└── InfoPanel (Panel)
    ├── PartName (Text)
    ├── PartDesc (Text)
    └── EquipButton (Button)
```

#### OutfitPanel 脚本配置：
```
【预制体】
- Category Tab Prefab: 分类标签预制体
- Part Item Prefab: 部件列表项预制体
- Preview Part Prefab: 预览部件预制体

【容器】
- Category Container: CategoryTabs
- Parts Container: PartsGrid
- Preview Container: PreviewArea

【显示】
- Part Name Text: InfoPanel/PartName
- Part Desc Text: InfoPanel/PartDesc
- Equip Button: InfoPanel/EquipButton
```

---

### 7. 主菜单"家园"按钮

在 `MainScene` 的 Canvas 中找到 `MainMenuPanel`：

```
1. 添加新按钮："家园"
2. 调整布局：建议放在"开始游戏"下方
3. 拖入 MainMenuPanel 的 Home Button 字段
```

## 🐛 常见问题

### Q: 进入家园场景后玩家不出现？
A: 检查 HomeManager 的 Player Prefab 和 Spawn Point 是否正确设置。

### Q: 点击NPC没有反应？
A: 检查：
1. HomeNPC 的 Outfit Panel 字段是否拖入了正确的UI
2. 玩家是否在交互半径内
3. OutfitPanel 脚本是否正确挂载

### Q: 装扮部件不显示在玩家身上？
A: 检查：
1. 玩家预制体是否有 PlayerOutfitApplier 组件
2. 部件的挂载点是否正确（Head/Body/Back）
3. 部件的 offset/scale 是否合适
