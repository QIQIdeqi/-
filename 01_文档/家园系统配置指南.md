# 家园系统配置指南

## 📋 系统概述

家园系统将原有的"皮肤系统"升级为"家园+装扮"复合系统：
- **主菜单** → 点击"家园" → 进入家园场景
- **家园场景** → 自由行走 + 装饰家园 + NPC换装
- **装扮系统** → 分部件换装（蝴蝶结、帽子、眼镜等）

## ⚙️ Unity配置步骤

### 1. 创建家园场景

1. 在 Unity 中点击 `File → New Scene`
2. 保存为 `Assets/Scenes/HomeScene.unity`
3. 在场景中创建：
   - **HomeManager** (空物体，挂载 HomeManager.cs)
   - **PlayerSpawnPoint** (空物体，标记玩家出生位置)
   - **HomeNPC** (Sprite，挂载 HomeNPC.cs)
   - **HomeDoor** (Sprite，挂载 HomeDoor.cs)
   - **Decorations** (多个装饰物，挂载 HomeDecoration.cs)
   - **Canvas** (UI画布，包含 OutfitPanel)

### 2. 配置 HomeManager

选中 HomeManager 物体，在 Inspector 中设置：
- **Main Menu Scene Name**: `MainScene` (你的主场景名)
- **Home Bounds**: 设置家园活动范围（如 Center: (0,0,0), Size: (10,10,0)）
- **Player Prefab**: 拖入玩家预制体
- **Player Spawn Point**: 拖入 PlayerSpawnPoint
- **Decorations**: 拖入所有装饰物
- **Home NPC**: 拖入 NPC 物体
- **Home Door**: 拖入门物体

### 3. 配置 HomeNPC

选中 NPC 物体：
- **Interaction Radius**: 1.5 (交互半径)
- **Interaction Key**: E (PC端交互按键)
- **Interaction Hint**: 拖入交互提示UI（如"按E对话"的文字）
- **Outfit Panel**: 拖入场景中的 OutfitPanel

### 4. 配置 HomeDoor

选中门物体：
- **Trigger Radius**: 1 (触发半径)
- **Arrow Indicator**: 拖入箭头UI（在门上显示的箭头）
- **Leave Hint**: 拖入离开提示文字
- **Leave Button**: 拖入离开按钮

### 5. 配置 OutfitPanel（装扮界面）

在 Canvas 下创建 OutfitPanel：
- 创建 Panel 作为主面板
- 创建分类标签容器（Horizontal Layout Group）
- 创建部件列表容器（Grid Layout Group）
- 创建预览区域（显示玩家和已装备的部件）
- 创建选中信息显示区域
- 拖入对应的预制体和按钮

### 6. 配置玩家预制体

选中玩家预制体：
- 添加 `PlayerOutfitApplier.cs` 组件
- 设置挂载点：HeadAttachment, BodyAttachment, BackAttachment

### 7. 配置主菜单

选中主菜单 Canvas：
- 添加"家园"按钮到 MainMenuPanel
- 将按钮拖入 MainMenuPanel 的 `Home Button` 字段

### 8. 配置 GameManager

选中 GameManager：
- 设置 `Home Scene Name` 为 `HomeScene`

## 🎨 创建装扮部件

### 方法1：使用编辑器工具
1. 在 Unity 菜单选择 `Geometry Warrior → 创建装扮部件`
2. 填写部件信息：
   - 部件ID (如: bow_red)
   - 部件名称 (如: 红色蝴蝶结)
   - 类别 (Bow/Hat/Glasses/...)
   - 部件精灵 (Sprite)
   - 图标 (可选)
   - 解锁条件
3. 点击"创建部件"

### 方法2：手动创建
1. 在 `Assets/Resources/OutfitParts/` 文件夹
2. 右键 → Create → Geometry Warrior → Outfit Part
3. 配置数据并保存

### 装扮部件类别
- **Bow** - 蝴蝶结
- **Hat** - 帽子
- **Glasses** - 眼镜
- **Scarf** - 围巾
- **Backpack** - 背包
- **Shoes** - 鞋子
- **Special** - 特殊装饰

## 🏡 创建装饰物

1. 在场景中创建 Sprite 物体
2. 添加 `HomeDecoration.cs` 组件
3. 配置：
   - Decoration Id: 唯一ID（用于保存位置）
   - Decoration Name: 显示名称
   - Category: 家具/植物/灯具等
   - Is Draggable: 是否可拖动
   - Can Rotate: 是否可旋转
   - Use Bounds: 是否限制在家园边界内

## 🔧 常见问题

### Q: OutfitManager 和 SkinManager 冲突吗？
A: 两者独立工作。OutfitManager 是分部件换装，SkinManager 是整体皮肤。可以同时存在，也可以逐渐用 OutfitManager 替换 SkinManager。

### Q: 如何只在主场景显示主菜单，在家园场景不显示？
A: GameManager 会自动处理。进入家园场景时状态变为 `Home`，主菜单会自动隐藏。

### Q: 装扮部件如何显示在玩家身上？
A: 确保玩家预制体有 `PlayerOutfitApplier` 组件，并正确设置了挂载点（Head/Body/Back）。

### Q: 如何添加新的部件类别？
A: 在 `OutfitPartData.cs` 中的 `OutfitCategory` 枚举添加新类别，然后在 `OutfitPanel.cs` 的 `GetCategoryName` 方法中添加对应的中文名。
