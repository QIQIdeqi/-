# 家园背包系统 UI 设计文档

## 📋 系统概述

家园背包系统是玩家管理家具库存的核心界面，需要兼顾功能性和女性向用户的审美需求。

---

## 🎨 视觉风格

### 整体风格
- **主题**: 手作毛毡/羊毛毡风格
- **色调**: 马卡龙 pastel 色系
- **质感**: 柔软、温暖、手工感

### 色彩系统

```
主色：     #FFB6C1 (浅粉色) - 按钮、高亮
辅助色：   #98D8C8 (薄荷绿) - 背景、选中状态
背景色：   #FFF5E6 (奶油白) - 面板背景
文字色：   #6B5B73 (紫灰) - 主要文字
次级文字： #9B8AA5 (浅紫灰) - 数量、提示
禁用色：   #D4C5D8 (灰紫) - 不可用时
```

### 质感设计
- 圆角半径：12-16px（柔软感）
- 阴影：柔和的弥散阴影（shadow: 0 4px 12px rgba(107, 91, 115, 0.15)）
- 边框：2px 手绘风格虚线或实线

## 🧩 组件设计

### 1. 标题栏

```
┌─────────────────────────────────┐
│  🏠  我的家具包              ✕  │
└─────────────────────────────────┘
```

- 左侧：小房子图标 + 标题
- 右侧：关闭按钮（圆形，带X图标）
- 背景：渐变粉色 #FFB6C1 → #FFC8D6

### 2. 分类标签

```
  ┌────────┐ ┌────────┐ ┌────────┐
  │  全部   │ │  桌子  │ │  椅子  │
  │ ●───── │ │ ────── │ │ ────── │
  └────────┘ └────────┘ └────────┘
     选中态     未选中    未选中
```

- **选中态**: 填充色 #98D8C8，文字白色，下方有小圆点指示器
- **未选中**: 透明背景，边框 #D4C5D8，文字 #9B8AA5
- 支持横向滑动（分类多时）
- 分类建议：全部、桌子、椅子、装饰、灯具、墙饰、地面

### 3. 家具列表项

```
      ┌─────────────┐
      │  ┌───────┐  │
      │  │       │  │  ← 家具图标 (72×72)
      │  │  🛋️   │  │     带柔和阴影
      │  │       │  │
      │  └───────┘  │
      │             │
      │  毛绒沙发    │  ← 家具名称
      │  ─────────  │
      │  持有: 2/3  │  ← 数量显示
      │  [放置 1个] │  ← 按钮（可选）
      └─────────────┘
        100×140px
```

#### 状态设计

| 状态 | 视觉表现 |
|------|----------|
| 正常 | 白色背景，正常图标，完整信息 |
| 选中 | 薄荷绿边框 #98D8C8，轻微放大 1.05x |
| 数量不足 | 整体变灰 50%透明度，文字显示"已获得: 0" |
| 已放置满 | 图标上叠加小房子图标，文字"场景中" |
| 按住拖拽 | 放大 1.1x，增加阴影，半透明 |

### 4. 数量显示

```
持有: 2/3  或  已获得: 0  或  场景中: 1
 ↑可用  ↑拥有            ↑未获得        ↑已放置

颜色:
- 绿色 #98D8C8: 可使用（available > 0）
- 灰色 #9B8AA5: 不可用
- 橙色 #FFB347: 已全部放置
```

### 5. 底部提示区

```
┌─────────────────────────────────┐
│  💡 长按家具拖拽到场景中摆放      │
│     或点击选择后点击场景位置      │
└─────────────────────────────────┘
```

- 背景：半透明奶油色 rgba(255, 245, 230, 0.9)
- 图标：小灯泡 💡
- 文字：#6B5B73，14px

## 🎯 交互设计

### 打开背包

1. 从屏幕底部滑入（0.3s ease-out）
2. 背景场景变暗（叠加 50% 黑色遮罩）
3. 播放轻快的"打开包裹"音效

### 选择家具

| 操作 | 响应 |
|------|------|
| 单击 | 选中该家具（边框高亮），显示可放置数量 |
| 双击 | 直接放置一个到上一次放置位置/屏幕中央 |
| 长按 | 进入拖拽模式，可拖到场景任意位置 |
| 滑动 | 滚动家具列表 |

### 拖拽放置

```
1. 长按家具 → 家具图标跟随手指
2. 拖到场景区域 → 显示家具虚影预览
3. 松开手指 → 家具放置到该位置
4. 播放"放置"音效（轻柔的"咚"声）
5. 自动关闭背包（可选设置）
```

### 分类切换

- 点击分类标签 → 家具列表淡入淡出切换（0.2s）
- 保持滚动位置记忆（每个分类独立）

### 关闭背包

- 点击关闭按钮 或 点击背景遮罩
- 向下滑出（0.25s ease-in）
- 背景恢复明亮

## 💻 实现建议

### Unity 组件结构

```
BackpackPanel (Canvas)
├── Background (Image - 半透明遮罩)
├── MainPanel (Image - 背包主面板)
│   ├── Header
│   │   ├── Icon (Image)
│   │   ├── Title (Text)
│   │   └── CloseBtn (Button)
│   ├── CategoryTabs (HorizontalLayoutGroup)
│   │   └── TabItem (Button) × N
│   ├── Content (ScrollRect)
│   │   └── GridLayoutGroup
│   │       └── FurnitureItemUI (Prefab) × N
│   └── Footer
│       └── HintText (Text)
└── DragPreview (Image - 拖拽预览，初始隐藏)
```

### 动画参数

```csharp
// 打开动画
mainPanel.DOAnchorPosY(0, 0.3f).SetEase(Ease.OutCubic);
background.DOFade(0.5f, 0.3f);

// 关闭动画
mainPanel.DOAnchorPosY(-Screen.height * 0.75f, 0.25f).SetEase(Ease.InCubic);
background.DOFade(0f, 0.25f);

// 选中动画
itemTransform.DOScale(1.05f, 0.15f).SetEase(Ease.OutBack);
```

### 响应式适配

```csharp
// 根据屏幕宽度调整列数
int columns = Screen.width < 600 ? 3 : 4;
gridLayout.constraintCount = columns;

// 网格单元大小自适应
float cellWidth = (Screen.width * 0.9f - padding) / columns - spacing;
gridLayout.cellSize = new Vector2(cellWidth, cellWidth * 1.4f);
```
