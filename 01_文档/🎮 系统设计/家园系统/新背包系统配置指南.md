# 新背包系统配置指南

## 概述

新的家园背包系统整合了**主角装扮**和**家园装扮**两个功能，通过左上角背包按钮进入。

## 功能流程

```
家园场景
    ↓
点击左上角背包按钮
    ↓
弹出背包面板（双Tab）
    ├─ Tab 1: 主角装扮
    │      └─ 点击部件切换装扮
    │
    └─ Tab 2: 家园装扮
           ├─ 显示所有已解锁家具
           ├─ 点击家具项显示"装饰"按钮
           └─ 点击"装饰"进入编辑模式
                  ↓
           家具悬浮在屏幕中央
           ├─ 拖拽移动位置
           ├─ 翻转按钮（如支持）
           ├─ 缩放滑条
           └─ 确认按钮
                  ↓
           确认后保存位置
           背包自动重新弹出
           回到之前的滚动位置
```

## 配置步骤

### 1. 创建家具数据

在 `Assets/Resources/Furniture/` 目录下创建家具数据：

1. **右键菜单** → `绒毛几何物语` → `家具数据`
2. 配置家具信息：
   - **家具ID**: 唯一标识（如 `chair_001`）
   - **家具名称**: 显示名称（如 "云朵沙发"）
   - **图标**: 背包中显示的小图标
   - **原尺寸图**: 场景中显示的完整Sprite
   - **默认缩放**: 1.0
   - **缩放范围**: 0.5 ~ 2.0
   - **是否可翻转**: 大部分家具支持

### 2. 创建背包按钮预制体

1. 在Canvas下创建按钮：`UI` → `Button`
2. 命名为 `BackpackButton`
3. 添加脚本：`BackpackButton.cs`
4. 配置：
   - 设置背包图标（Sprite）
   - 拖入 `BackpackPanel` 预制体
5. 保存为预制体到 `Assets/Prefabs/UI/`

### 3. 创建背包面板预制体

1. 在Canvas下创建面板：`UI` → `Panel`
2. 命名为 `BackpackPanel`
3. 添加脚本：`BackpackPanel.cs`
4. 配置结构：

```
BackpackPanel (Panel)
├── 标题栏 (Text: "背包")
├── 关闭按钮 (Button)
├── Tab容器 (Horizontal Layout)
│   ├── 主角装扮Tab (Button)
│   └── 家园装扮Tab (Button)
├── 主角装扮内容 (Scroll View)
│   └── Content (Grid Layout)
│       └── OutfitItem预制体
└── 家园装扮内容 (Scroll View)
    └── Content (Grid Layout)
        └── FurnitureItem预制体
```

5. 创建家具项预制体 `FurnitureItemUI`：
   - Icon (Image)
   - Name (Text)
   - SelectedBorder (Image)
   - DecorateBtn (Button)

6. 保存为预制体到 `Assets/Prefabs/UI/`

### 4. 创建家具编辑控制器预制体

1. 在Canvas下创建空物体：`FurnitureEditController`
2. 添加脚本：`FurnitureEditController.cs`
3. 配置工具栏UI：

```
FurnitureEditController
└── ToolbarPanel (Panel - 锚点: Center)
    ├── FlipBtn (Button)
    ├── ScaleSlider (Slider)
    │   └── ScaleValueText (Text)
    └── ConfirmBtn (Button)
```

4. 保存为预制体到 `Assets/Prefabs/UI/`

### 5. 配置HomeManager

在 `HomeScene` 中的 `HomeManager` 上配置：

| 字段 | 说明 |
|------|------|
| 背包按钮预制体 | 拖入 `BackpackButton.prefab` |
| 背包按钮父节点 | 留空（自动使用Canvas）或指定 |
| 家具编辑控制器预制体 | 拖入 `FurnitureEditController.prefab` |

### 6. 配置家具解锁

家具数据默认 `isUnlocked = true`，可以通过以下方式控制：

```csharp
// 解锁家具
furnitureData.isUnlocked = true;

// 检查是否解锁
if (furnitureData.isUnlocked) {
    // 显示在背包中
}
```

### 7. 从存档加载已解锁家具

修改 `BackpackPanel.cs` 中的 `LoadFurnitureData()`：

```csharp
private void LoadFurnitureData()
{
    allFurniture = new List<FurnitureData>();
    
    // 从存档加载已解锁家具ID列表
    string[] unlockedIds = SaveManager.Instance.GetUnlockedFurnitureIds();
    
    foreach (var id in unlockedIds)
    {
        var furniture = Resources.Load<FurnitureData>($"Furniture/{id}");
        if (furniture != null)
        {
            allFurniture.Add(furniture);
        }
    }
}
```

## 美术资源需求

### 家具图标
- 尺寸：100x100
- 风格：羊毛毡手作风格
- 背景：透明

### 背包UI
- 背包按钮图标：64x64
- 面板背景：圆角矩形，pastel色调
- Tab按钮：选中态/未选中态

### 家具原尺寸图
- 按实际尺寸设计
- 支持翻转（左右对称或设计翻转版本）
- 风格统一：羊毛毡手作感

## 保存数据格式

家具位置保存在 `PlayerPrefs`：

```
Key: HomeDecorations_{furnitureId}
Value: "x,y,z,scale,isFlipped"
```

示例：
```
HomeDecorations_chair_001 = "1.5,2.0,0,1.2,false"
```

## 注意事项

1. 家具编辑时玩家移动会被禁用
2. 确认后家具位置和状态才会保存
3. 取消编辑会直接销毁临时家具实例
4. 背包会自动记录滚动位置，重新打开时恢复
