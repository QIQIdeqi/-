# 暂停按钮设计提示词 - 绒毛几何物语

## 设计理念
- **风格**：羊毛毡手作、纽扣、刺绣
- **质感**：柔软、温暖、有手工温度
- **识别度**：清晰的暂停符号（||）
- **尺寸**：适合移动端点击（建议80x80px以上）

---

## 方案1：毛毡纽扣风格（推荐）

### 描述
将暂停按钮设计成一个大号的手工毛毡纽扣，中心绣有暂停符号。

### 英文提示词
```
A cute game UI button for "pause", isolated on transparent background.

Main Shape:
- A large round button made of thick wool felt
- Diameter: Perfect circle, plush and puffy appearance
- Thickness visible from subtle side shadow, like a stuffed cushion

Material & Texture:
- Surface: Soft fuzzy wool felt texture, visible fiber details
- Color: Warm pastel pink #FFB6C1 or mint green #98FF98
- Edge: Thick visible edge with blanket stitch embroidery in cream thread
- The felt looks slightly uneven, handmade quality

Pause Symbol:
- Two vertical parallel lines (pause icon) embroidered in center
- Stitch style: Running stitch or satin stitch
- Thread color: Darker shade of the button color, or cream/white
- Lines have slightly rounded ends, not sharp
- Size: About 40% of button diameter, well-centered

Details:
- Four small thread holes at button corners (decorative, like a real button)
- Holes have white or cream thread passing through
- Soft drop shadow beneath the button for depth
- Tiny felt sparkles or dots around the edge as decoration
- Optional: Small felt flower or star accent near the edge

Lighting & Style:
- Soft diffused lighting from top-left
- Warm, cozy atmosphere
- Kawaii aesthetic, extremely cute
- High quality game asset, clean edges

Parameters: --ar 1:1 --style raw --v 6.0 --no background, shadow
Negative: sharp edges, metallic, plastic shine, neon, dark colors, realistic photography, geometric hard lines
```

### 中文描述
一个可爱的游戏UI暂停按钮，透明背景。

主形状：
- 一个大号圆形纽扣，由厚羊毛毡制成
- 直径：完美的圆形，蓬松饱满的外观
- 从侧面阴影可见厚度，像一个小抱枕

材质与纹理：
- 表面：柔软毛茸茸的羊毛毡纹理，可见纤维细节
- 颜色：温暖的马卡龙粉 #FFB6C1 或薄荷绿 #98FF98
- 边缘：厚实的可见边缘，带有奶油色线的毯子刺绣
- 毡布看起来略微不平整，手工品质

暂停符号：
- 中心刺绣的两条垂直平行线（暂停图标）
- 针法：跑针或缎纹针
- 线色：比纽扣颜色深一点的色调，或奶油/白色
- 线条有略微圆润的末端，不尖锐
- 大小：约占纽扣直径的40%，居中

细节：
- 纽扣四角的四个小线孔（装饰性，像真纽扣）
- 孔洞有白色或奶油色线穿过
- 按钮下方有柔和的投影增加深度
- 边缘周围有小毡闪光或圆点作为装饰
- 可选：边缘附近有小毡花或星星点缀

光线与风格：
- 左上方的柔和漫射光
- 温暖、舒适的氛围
- 可爱美学，极其可爱
- 高质量游戏资源，干净的边缘

```
- Handmade craft aesthetic
- Cozy and inviting
- Soft lighting, warm tones
- Kawaii, storybook quality
```

## 方案5：极简毛毡风格（备用状态）

### 描述
暂停和播放两种状态的按钮设计。

### 暂停状态
```
A minimalist game UI pause button, isolated on transparent background.

Shape:
- Simple circle or rounded square
- Soft felt material appearance

Design:
- Two vertical parallel bars in center
- Bars are slightly rounded at ends
- Color: Soft pastel (pink, blue, or mint)
- Simple, clean, cute

Style:
- Flat design with subtle felt texture
- Soft shadow
- Very clean edges

Parameters: --ar 1:1 --style raw --v 6.0 --no background
```

### 播放状态（配套）
```
A minimalist game UI play button matching the pause style, isolated on transparent background.

Shape:
- Same shape and size as pause button
- Same felt material and color

Design:
- Triangle play icon pointing right
- Rounded corners on triangle
- Same style as pause bars

Style:
- Consistent with pause button
- Soft felt texture
- Clean and simple

Parameters: --ar 1:1 --style raw --v 6.0 --no background
```

## 配套设计：按下状态

```
A pressed/clicked version of the felt button pause icon, isolated on transparent background.

Changes from normal state:
- Slightly darker overall tone (pressed lighting)
```

## Unity UI设置建议

### 按钮组件设置
```csharp
// Button组件配置
Button pauseButton;
Image buttonImage;

// 推荐尺寸
RectTransform size: 80x80px to 100x100px (移动端)

// 颜色过渡
Normal Color:     White (1, 1, 1, 1)
Highlighted Color: Light Pink (1, 0.9, 0.9, 1) 或稍微放大
Pressed Color:     Slightly darker (0.9, 0.9, 0.9, 1)
Disabled Color:    Gray (0.5, 0.5, 0.5, 0.5)
```

### Sprite导入设置
1. **Texture Type**: Sprite (2D and UI)
2. **Sprite Mode**: Single
3. **Pixels Per Unit**: 100
4. **Generate Mip Maps**: Off (UI不需要)

### 尺寸建议
| 用途 | 推荐尺寸 | 备注 |
|-----|---------|------|
| 手机端 | 80x80px | 最小可点击区域 |
| 平板 | 100x100px | 更大更易点击 |
| 菜单界面 | 60x60px | 次级功能 |
