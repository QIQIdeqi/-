### 3. 子弹系统调试

#### 问题
Projectile（子弹）生成后在场景中看不到

#### 调试过程
1. 添加大量 Debug.Log 追踪生成流程
2. 发现 SpriteRenderer 的 Sprite 为 null
3. 发现 ProjectileVisualManager 的 Default Material 未配置

#### 最终解决
在团结引擎中配置：
- `ProjectileVisualManager` 的 Default Material 和 Default Shape
- 确保子弹预制体有 SpriteRenderer 且 Sprite 不为空

### 4. 子弹方向修正

**问题**：子弹发射时 Sprite 旋转 90°
**解决**：在 `Projectile.cs` 的 `FixedUpdate` 中调整角度计算
```csharp
float angle = Mathf.Atan2(direction.y, direction.x) * Mathf.Rad2Deg - 90f;
```

### 5. 虚拟摇杆实现

#### 创建文件
- `Assets/Scripts/UI/Joystick.cs` - 虚拟摇杆控制

#### 功能
- 支持触摸和鼠标输入
- 固定位置模式（默认）
- 把手只在背景圆盘范围内移动
- 输出标准化方向向量

#### 使用方式
在 `PlayerController` 中添加：
```csharp
public Joystick joystick;
// 在 HandleInput 中优先使用摇杆输入
if (joystick != null && joystick.IsDragging)
{
    moveInput = joystick.GetDirection();
}
```

### 8. 暂停功能实现

#### 修改文件
- `Assets/Scripts/UI/GameOverPanel.cs` - 支持两种模式
- `Assets/Scripts/Manager/GameManager.cs` - 添加 PauseGame/ResumeGame
- `Assets/Scripts/UI/HUDPauseButton.cs` - 暂停按钮（新建）

#### 功能
GameOverPanel 现在支持两种模式：
- **GameOver 模式**（死亡）：标题"游戏结束"，按钮"复活（看广告）"
- **Paused 模式**（暂停）：标题"游戏暂停"，按钮"继续游戏"

#### 使用
```csharp
// 暂停游戏
GameManager.Instance.PauseGame(); // 显示"游戏暂停"面板，Time.timeScale = 0

// 继续游戏
GameManager.Instance.ResumeGame(); // Time.timeScale = 1
```

### 9. 其他修复

#### Time.timeScale 问题
**问题**：游戏开始后时间暂停，角色不动
**原因**：之前升级界面暂停后未恢复，或异常退出导致
**解决**：在 `StartGame()` 和 `UpgradeManager.Start()` 中强制设置 `Time.timeScale = 1f`

#### GameManager 初始化
**问题**：MainMenuPanel 中获取不到 GameManager 实例
**原因**：Awake 执行顺序问题
**解决**：改为在 Start 中获取
